## 🚀 Getting Started

### Prerequisites
Ensure you have [Node.js](https://nodejs.org) (v16 or higher) installed on your system environment.

### Installation
1. Clone or download this project codebase repository locally.
2. Initialize your local terminal configuration to point directly at the root folder path.
3. Install the application runtime dependencies:
   ```bash
   npm install
   ```

### Execution
Launch the application server inside your developer terminal panel using the start script instruction:
```bash
npm start
```
Once initialized, navigate to your web browser window and open: `http://localhost:3000`

## 📁 Repository Map
* `public/index.html` - Main Dark-Theme UI and currency dashboard layout structure.
* `public/styles.css` - Custom styling layout properties matching modern platform palettes.
* `src/fxService.js` - Settlement arithmetic calculations from base currency to local Thai Baht (THB).
* `src/server.js` - Main system routing network powered by Node and Express framework layers.
